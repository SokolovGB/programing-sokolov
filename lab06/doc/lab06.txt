Знайти відношення максимального елементу масиву до мінімального. 
Врахувати, що ділити на 0 не можна.

1. Введення розміру масиву:
Програма розпочинається із введення цілочисельної змінної n за допомогою функції scanf.
Якщо n менше або дорівнює 0, програма виводить повідомлення про некоректний розмір масиву 
і завершує виконання із кодом повернення 1.

2. Виділення пам'яті для масиву:
Якщо n додатнє, програма створює масив arr розміром n.

3. Введення елементів масиву:
Відбувається цикл, в якому пропонується ввести значення елементів масиву.
Значення зберігаються в масиві arr.

4. Перевірка розміру масиву:
Якщо розмір масиву n рівний 1, програма виводить повідомлення про те, 
що масив складається з одного елемента, і завершує виконання із кодом повернення 0.

5. Знаходження максимального та мінімального елементів:
Ініціалізація змінних max та min значеннями першого елемента масиву.
Відбувається цикл, де кожний елемент порівнюється із поточними значеннями max та min,
і якщо елемент більший за max або менший за min, відповідні змінні оновлюються.

6. Розрахунок та вивід відношення максимального до мінімального:
Якщо мінімальний елемент рівний 0, програма виводить повідомлення про неможливість ділення на 0.
У протилежному випадку програма розраховує відношення максимального до мінімального елемента 
і виводить результат із точністю до двох знаків після коми.

7. Завершення програми:
Програма завершує виконання із кодом повернення 0.

Код програм:

#include <stdio.h>

int main() {
    int n;
    printf("Введіть розмір масиву: ");
    scanf("%d", &n);

    if (n <= 0) {
        printf("Некоректний розмір масиву.\n");
        return 1;
    }

    int arr[n];

    printf("Введіть елементи масиву:\n");
    for (int i = 0; i < n; i++) {
        scanf("%d", &arr[i]);
    }

    if (n == 1) {
        printf("Масив складається з одного елементу. Немає відношення.\n");
        return 0;
    }

    int max = arr[0];
    int min = arr[0];

    for (int i = 1; i < n; i++) {
        max = (arr[i] > max) ? arr[i] : max;
        min = (arr[i] < min) ? arr[i] : min;
    }

    if (min == 0) {
        printf("Ділення на 0 неможливе.\n");
    } else {
        float ratio = (float)max / min;
        printf("Відношення максимального до мінімального: %.2f\n", ratio);
    }

    return 0;
}

Функція main()C є обов'язковою точкою входу до програми. Вона виконує роль стартової функції, з якої починається виконання програми після її запуску.
