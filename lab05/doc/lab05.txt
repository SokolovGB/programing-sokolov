У заданому діапазоні цілих чисел визначити останнє число, що ділиться на 7 без залишку

Основна частина коду
1. Оголошення змінних:
int start, end; Вони будуть використовуватися для зберігання початкового та кінцевого значень діапазону
2. Введення початку та кінця діапазону:
 за допомогою printf-Введення початку та кінця діапазону, за допомогою scanf введені значення зберігаються в змінних start та end.
3. Знаходження останнього числа, що ділиться на 7 без залишку:
int lastMultipleOf7 = end - (end % 7)
 Використовуєте арифметичний вираз, щоб знайти останнє число в діапазоні, яке ділиться на 7 без залишку. end % 7 повертає залишок від ділення end на 7, а потім ви віднімаєте цей залишок від end, отримуючи останнє число, яке ділиться на 7.
4. Виведення результату на екран за допомогою printf
5. Повернення 0 для завершення програми за допомогою return


Код програм:
#include <stdio.h>

int main() {
    int start, end;

    // Запит користувача для введення початку та кінця діапазону
    printf("Введіть початок діапазону: ");
    scanf("%d", &start);

    printf("Введіть кінець діапазону: ");
    scanf("%d", &end);

    // Знаходження останнього числа, що ділиться на 7 без залишку
    int lastMultipleOf7 = end - (end % 7);

    // Виведення результату
    printf("Останнє число в діапазоні, що ділиться на 7 без залишку: %d\n", lastMultipleOf7);

    return 0;
}

Функція main()C є обов'язковою точкою входу до програми. Вона виконує роль стартової функції, з якої починається виконання програми після її запуску.

Виконуєм зупинку посередині виконання програми за допомогою breakpoints. Приклад зміни стану програми “на льоту”
Точка зупинки 
- b 12
Встановлюємо точку зупинки, перед перевіркою умови знаходження останнього числа, що ділиться на 7 без залишку. 
Після вводу дiапазону чисель, змінюємо на інше значення командою expr end = end - (end % 5) умову знаходження останнього числа, що ділиться на 5 без залишку
Получаемо значення. Завершуем программу з попередньою умовою.
